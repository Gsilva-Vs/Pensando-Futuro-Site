###  Sobre o projeto

**Pensando Futuro** é um site educacional que oferece conteúdo gratuito e de qualidade para estudantes que desejam ser aprovados nos principais vestibulares do Brasil (ENEM, Fuvest, Unicamp, etc.). O projeto une design moderno, performance e uma experiência pensada para facilitar a jornada de estudos.

---

###  Funcionalidades

-  **Guias completos** sobre cada vestibular (formato, disciplinas, peso das provas)
-  **Dicas de estudo** e planejamento de rotina eficiente
-  **Notas de corte** das principais universidades (com gráficos dinâmicos)
-  **Ajuda na escolha** de curso e faculdade
-  **Materiais úteis** e curadoria de conteúdos gratuitos
-  **Design responsivo** – funciona perfeitamente em desktop, tablet e mobile

###  Design System

O projeto segue um Design System próprio, garantindo consistência visual e facilidade de manutenção.

- **Paleta de cores principais:**  
  Roxo primário (`#9333ea`) e Amarelo de destaque (`#fbbf24`)
- **Tipografia:** Fonte `Inter` (excelente legibilidade)
- **Espaçamento:** Grid baseado em 8px para harmonia entre elementos
- **Componentes:** Botões, cards e formulários padronizados reutilizáveis

> O Design System completo (cores, tipografia, tamanhos, variantes) está documentado no arquivo [`design-system.md`](./design-system.md) na raiz do projeto.
